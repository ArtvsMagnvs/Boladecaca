# Material crudo JWIKI-001 — Historia cronológica 1990s-2026

## Hechos verificados

1. **1997 — Microsoft Clippy (Office Assistant)** — Primer asistente mainstream integrado en software. Basado en reglas, no en IA real. Apagado en 2007.
   - Fuente: Wikipedia + blogs Microsoft
2. **2001 — SmarterChild** — Bot de chat en AOL Instant Messenger y MSN Messenger. 30M+ usuarios en 2002. Cerrado en 2008.
   - Fuente: industry retrospectives
3. **2011 — Apple Siri** — Primer asistente de voz mainstream en smartphone (iPhone 4S).
   - Fuente: Apple keynote + reviews
4. **2012 — Google Now** — Asistente contextual de Google.
   - Fuente: Google blog
5. **2014 — Amazon Alexa / Echo** — Primer altavoz inteligente con wake word. Lanzado noviembre 2014.
   - Fuente: Amazon press release
6. **2016 — Google Assistant** — Sucesor de Google Now.
   - Fuente: Google IO 2016
7. **2018 — OpenAI GPT-1** — Paper publicado. 117M parámetros.
   - Fuente: OpenAI paper
8. **2019 — GPT-2** — 1.5B parámetros. OpenAI inicialmente no lo libera por miedo a uso malicioso.
   - Fuente: OpenAI blog
9. **2020 — GPT-3** — 175B parámetros. Lanzado vía API. Inicia era de LLMs comerciales.
   - Fuente: OpenAI API launch
10. **2022-Nov — ChatGPT** — Lanzamiento público. 1M usuarios en 5 días, 100M en 2 meses (récord absoluto).
    - Fuente: OpenAI + Reuters
11. **2023-Mar — GPT-4** — Multimodal. 5x más capaz que GPT-3.5.
    - Fuente: OpenAI
12. **2023-Nov — AutoGen v0.2 (Microsoft)** — Primer framework prominente de multi-agent.
    - Fuente: Microsoft Research
13. **2024-Ene — CrewAI launch** — Framework multi-agent basado en roles (crews + tasks).
    - Fuente: CrewAI repo
14. **2024-Jul — LangGraph v0.1 (LangChain)** — Framework para stateful agent workflows.
    - Fuente: LangChain blog
15. **2025-Mar — OpenClaw v0.5 (openclaw/openclaw)** — Primer release público. TypeScript-first. Discord/Telegram/WhatsApp/Slack.
    - Fuente: GitHub releases
16. **2025-Oct — JarvisAgent v1.0 (myismu/JarvisAgent)** — Tauri 2 + Vue 3 + Rust. Multi-LLM.
    - Fuente: GitHub
17. **2026-Ene — Hermes Agent v0.8.0 (Nous Research)** — 53k stars, self-evolving.
    - Fuente: GitHub
18. **2026-Mar — OpenHuman v0.50** — Rust + TS, desktop-first. Tinyhumansai.
    - Fuente: GitHub
19. **2026-Abr — OpenClaw 354k stars** — Se convierte en el repo AI assistant #1 por stars.
    - Fuente: GitHub API
20. **2026-May — OpenJarvis (Stanford) doc oficial** — Framework local-first con 5 primitivos.
    - Fuente: open-jarvis.github.io
21. **2026-Jun — OpenClaw 376k stars** — Líder absoluto. Superpowers 215k stars (skill framework).
    - Fuente: GitHub trending

## Snippets de código

- OpenClaw: TypeScript, `package.json` con 50+ integraciones. Repo: github.com/openclaw/openclaw
- Hermes Agent: Python + Node.js, 53k stars, MIT, self-hosted.
- OpenHuman: Rust backend + TS frontend, 7.8k stars, v0.53.43 (mayo 2026).
- OpenJarvis: Python, local-first, 5 primitivos (model, reasoning, agent, tools/memory, routing).
- JarvisAgent: Tauri 2 + Vue 3 + Rust, 20+ LLMs.

## Diferencias entre proyectos

- **Local-first**: OpenJarvis (Python), OpenHuman (Rust), JarvisAgent (Rust)
- **Multi-platform cloud**: OpenClaw (TS, Discord/Telegram/Slack/WhatsApp)
- **Skill framework**: Superpowers (Shell, 215k)
- **MCP-based**: Clawdbot
- **Coding focus**: JarvisAgent (dual-axis mode Audience x WorkMode)
- **Multi-modal**: Hermes Agent (Nous Research)
