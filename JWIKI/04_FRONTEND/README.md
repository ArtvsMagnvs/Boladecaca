## Aithera stack

- **React 18**: libreria UI.
- **TypeScript 5.3**: type safety.
- **Vite 5**: build tool.
- **Electron 29**: desktop wrapper.
- **HashRouter**: routing (necesario para `file://`).
- **Zustand 4**: state.
- **Tailwind 3.4**: CSS.
- **Framer Motion 11**: animaciones.
- **@react-three/fiber + three.js 0.160**: 3D (AICore).
- **@react-three/drei**: helpers 3D.

## Por qué React

- ✅ Ecosistema masivo.
- ✅ TypeScript first-class.
- ✅ React Three Fiber (3D).
- ✅ Contratar devs fácil.

## Alternativas

- **Vue 3**: Composition API excelente, pero ecosistema AI/3D menor.
- **Svelte 5**: compile-time, muy rápido, pero menos componentes 3D.
- **SolidJS**: React-like con mejor performance, pero menos maduro.

## Para Aithera

React es la elección correcta (3D, AI, ecosystem).
