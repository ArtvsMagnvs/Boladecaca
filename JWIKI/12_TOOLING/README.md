# 12 — TOOLING

> Execution engines, tool managers, validators, file/shell/git/email/calendar tools.

## Alcance

- Execution engines: validación + ejecución controlada (en uso en Aithera)
- Tool managers: registro centralizado, whitelist por agente (ToolManager en Aithera)
- Validadores: path traversal, command injection, JSON schema
- Filesystem tools: read/write/list, sandboxed paths
- Shell tools: comando whitelist (python, git, npm, uvicorn)
- Git tools: status, log, diff, commit
- Email/Calendar tools: Gmail REST, Google Calendar (en uso en Aithera)
